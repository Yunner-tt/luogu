# 本文档为字符串练习总结文档
> ## 字符串读取
>> ### getchar()
>> 该函数意思为：读取并拿走，一般可以这样使用：<br>`while ((tmp = getchar())!='\n')`
>> 代表逐字符拿取并读出以行为单位

>> ### gets()
>> 该函数配合char数组使用，读取一行直到换行(空格也读):<br>
>> `char a[10]`<br>`gets(a)`<br>
>> 这样通过遍历数据也能达到逐个读取的目的

>> ### getline()
>> 该函数配合string使用，读取一行直到换行(空格也读):<br>
>> `string ss`<br>`getline(cin,ss)`<br>
>> eg:string类型是有[ ]操作符的，跟数组意义一样
